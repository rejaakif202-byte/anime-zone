Telegram Links Guide
Season-wise:
Agar ek poore season ke episodes ek Telegram post/folder mein hain — New Season banate time "Season Telegram Links" mein links add karo. Episode Range daalo (e.g. 1-26) taaki button pe range dike.

Episode-wise:

Smart Display Logic:
• Video URL diya + TG link diya → dono show honge
• Sirf Video URL diya → sirf player dikhega
• Sirf TG link diya → thumbnail + TG buttons dikhenge

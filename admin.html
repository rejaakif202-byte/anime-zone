Video Hosting Guide

Catbox.moe — BEST (Free, Direct Play, No Account)
1. Go to catbox.moe
2. Click "Choose Files" → select your video
3. Click "Upload" (supports up to 200MB, no account needed)
4. Copy the link (ends in .mp4) → Paste in Video URL
✅ Video plays directly in the built-in player! Full controls!
Streamtape.com — Good (Free, Unlimited Size)
1. Register at streamtape.com (free)
2. Upload video → Go to file page
3. Copy the video URL (streamtape.com/v/...)
4. Paste in Video URL field
✅ Plays as embedded player. No file size limit!
YouTube (Unlisted)
Upload → Visibility: Unlisted → Share → Copy link → Paste in Video URL
✅ Works great. No file size limit!
Google Drive — ⚠️ Often Blocked
Google Drive videos often fail to load because Google blocks embedding on external sites.
❌ Not recommended. Use Catbox or Streamtape instead!
GoFile.io — Works (Free, Large Files)
Upload video → Get direct download link
Use the "direct download" URL (not the page URL)
✅ Works for direct playback if you get the raw file URL!
When to use Telegram Links
Season Pack: Add Telegram folder/post link for the full season. Shows on anime detail page only.

Episode TG Link: Add if you have NO video URL for that episode. If video URL is present → Telegram shown alongside video.

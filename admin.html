📖 Upload Guide

YouTube (Easiest)
1. Open YouTube Studio
2. Upload video
3. Visibility → Unlisted (NOT public)
4. Share → Copy link
5. Paste in admin panel ✅
Google Drive
1. Upload video to Drive
2. Right click → Share → Anyone with link
3. Copy link → Paste in admin panel ✅
Telegram (Bulk Files)
Since you have anime files in Telegram:

Option 1 — Bot Direct Link:
1. Forward files to @filetolink_bot on Telegram
2. Bot will give you a direct streamable link
3. Paste that link in admin panel ✅

Option 2 — Streamtape (Free):
1. Go to streamtape.com
2. Upload video
3. Get embed link → Paste ✅

Option 3 — YouTube Bulk:
Use YouTube Studio → Upload — you can upload multiple videos. Set all as Unlisted. This is the most reliable for streaming.
💡 For Thumbnails:
• Google Images → Right click image → Copy image address
• Or upload to Google Drive → Share → Copy link

📖 Video Upload Guide

YouTube (Sabse Aasaan)
1. YouTube app kholo
2. Upload button (+) dabao
3. Video select karo
4. Title daalo
5. Visibility → "Unlisted" select karo (Public mat karo!)
6. Upload karo
7. Video open karo → Share → Copy Link
8. Woh link admin panel mein paste karo ✅
Google Drive
1. drive.google.com kholo
2. "+ New" → File Upload → Video select karo
3. Upload hone do (time lagega)
4. Video pe long press → Share
5. "Anyone with the link" select karo
6. "Copy link" dabao
7. Woh link admin panel mein paste karo ✅
Telegram (Best for Large Files)
1. Telegram pe private channel banao
2. Video upload karo (2GB tak free)
3. Message pe press hold karo → Copy Link
4. Woh link admin panel mein paste karo ✅

⚠️ Note: Telegram direct stream nahi karta — YouTube ya Drive better hai.
